# HAL — Natural Language Synthesizer

HAL is an AI-powered polyphonic subtractive synthesizer. Describe a sound in plain English — "warm analog pad with slow attack", "punchy 808 sub bass", "gritty supersaw lead" — and HAL generates a complete synth patch instantly using Claude.

Built with [iPlug2](https://github.com/iPlug2/iPlug2) for macOS (arm64 + x86_64). Available as VST3, AU, and Standalone App.

---

## Features

- **Natural language patch generation** — type any sound description, get a full patch in ~1–2 seconds
- **Nudge/refine mode** — iterate on a generated patch with follow-up instructions ("make it darker", "add more reverb")
- **3-oscillator engine** — sine, saw, square, triangle; up to 8-voice unison per oscillator
- **SVF filter** — LP12, LP24, HP12, BP, Notch with filter envelope and key tracking
- **Pitch envelope** — one-shot pitch sweep for 808 kicks and stabs
- **2 LFOs** — 4 waveform shapes, 4 modulation targets (filter cutoff, pitch, volume, pan)
- **Effects** — stereo chorus, Freeverb reverb, feedback delay, EQ shelves, soft saturation
- **Arpeggiator** — up/down/updown/random patterns, 1–4 octaves
- **"Not ___" preset bank** — 10 curated presets inspired by classic synths (Not Minimoog, Not TB-303, Not 808, etc.)
- **WAV export** — 24-bit stereo with SMPL loop chunk, ready for samplers
- **Vital export** — generate a `.vital` preset file from any patch

## Building

```bash
# Clone iPlug2 as a sibling directory
git clone https://github.com/iPlug2/iPlug2.git ../iPlug2

# Build the standalone app (fastest for development)
xcodebuild -project HAL/projects/HAL-macOS.xcodeproj -target APP -configuration Debug

# Build VST3
xcodebuild -project HAL/projects/HAL-macOS.xcodeproj -target VST3 -configuration Debug

# Open the app
open ~/Applications/HAL.app
```

The VST3 builds to `~/Library/Audio/Plug-Ins/VST3/` automatically.

## How to Use

### Generating a patch

1. Type a sound description in the **DESCRIBE** field (e.g. "dark reese bass with slow filter movement")
2. Optionally select a **category tab** (PAD / ARP / BASS / LEAD / STAB / 808 / FX) — this biases the generation
3. Press **GENERATE** — the AI returns a full patch in 1–2 seconds and applies it live

### Refining a patch

1. Type a follow-up instruction in the **NUDGE** field (e.g. "brighter" or "more aggressive attack")
2. Press **GENERATE** — the current patch is sent along with the nudge for targeted adjustment

### Using presets

Click **Choose...** in the bottom-left panel to load any of the 10 "Not ___" presets as a starting point.

### Exporting

- **WAV** — renders a 5-second sample (3s hold + 2s tail) with SMPL loop points to `~/Desktop/`
- **VITAL** — exports a `.vital` preset file to `~/Desktop/`, importable directly into Vital
